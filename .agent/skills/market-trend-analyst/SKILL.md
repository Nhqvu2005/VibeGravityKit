# Market Trend Analyst

## Purpose
Identifies current market trends and "must-have" features for a specific domain to ensure the product remains competitive and modern.

## Usage

### 1. Analyze Trends
```bash
python .agent/skills/market-trend-analyst/scripts/trends.py --industry "edtech"
```

### Output
- Top 5 Trending Features.
- Rising Technologies (e.g., AI, VR).
- User Expectations (e.g., Mobile-first, Dark mode).
- Declining Trends to avoid.
