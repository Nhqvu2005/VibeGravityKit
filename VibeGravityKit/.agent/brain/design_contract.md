# Design Contract: VibeGravityKit Docs

## Colors
- **Primary**: #3B82F6 (Blue 500) - Links, Accents
- **Secondary**: #64748B (Slate 500) - Muted text
- **Background (Light)**: #FFFFFF (White)
- **Background (Dark)**: #0F172A (Slate 900)
- **Surface (Light)**: #F8FAFC (Slate 50)
- **Surface (Dark)**: #1E293B (Slate 800)
- **Text (Light)**: #0F172A (Slate 900)
- **Text (Dark)**: #F8FAFC (Slate 50)
- **Border**: #E2E8F0 (Slate 200) / #334155 (Slate 700)

## Typography
- **Heading Font**: Inter, sans-serif (Google Fonts)
- **Body Font**: Inter, sans-serif (Google Fonts)
- **Code Font**: 'JetBrains Mono', monospace
- **Font Sizes**: base: 16px, lg: 18px, xl: 24px, 2xl: 30px

## Component Rules
- **Buttons**:
  - Style: Rounded (6px), solid color (Primary) or outline (Border).
  - Hover: Opacity 0.9 or slight background shift.
- **Cards**:
  - Style: Flat with subtle border (1px solid var(--border)). No heavy shadows.
- **Navigation**:
  - Style: Sticky top bar. Glassmorphism effect (backdrop-filter: blur(8px)).
- **Theme Toggle**:
  - Style: Icon button (Sun/Moon).
- **Markdown Content**:
  - Max-width: 800px (prose).
  - Line-height: 1.6.

## Layout
- **Max Width**: 1200px (Container)
- **Responsive**: Mobile-first. Padding 1rem on mobile, 2rem on desktop.
